### 选择github blog的几个原因：
1. 让自己的github内容更充实，首页的contributions图更好看
2. 界面简洁，更容易让自己和读者注重blog的内容和质量，而不是主题的好看和花哨
3. issue的评论可以和别人更好的交流
4. 作为程序员的门面和工作台，每天花更多的时间在github上，有利无弊
